neurology Keterangan Model

Ringkasan cara kerja prediksi dan evaluasi arah

Sistem ini menggunakan model Long Short-Term Memory (LSTM) untuk memprediksi harga penutupan emas mingguan. Data harga diambil dari GC=F melalui yfinance, lalu fitur harga dibentuk dalam rasio terhadap close minggu sebelumnya agar model lebih fokus pada pola perubahan, bukan angka harga absolut.

Input Harga

OHLC mingguan GC=F, close minggu lalu, dan close dua minggu lalu.

Variabel Makro

Federal Funds Rate, perubahan FFR, dan Non-Farm Payroll.

Target Model

Model mempelajari rasio y_ratio = target close / close_lag1.

Evaluasi Web

Status benar/salah dihitung dari arah close minggu kemarin (lag1) menuju actual close minggu berikutnya.

Catatan: hasil prediksi menunjukkan estimasi arah dan harga penutupan mingguan berdasarkan data historis serta variabel makro yang digunakan model. Evaluasi arah pada tabel backtest dibuat agar lebih mudah dibaca secara praktis, yaitu membandingkan prediksi dan aktual terhadap close minggu berjalan.
